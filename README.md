## Running the project
### Local setup
To run this project locally: 
1. Clone the project locally, change into the directory, and install the dependencies:

```sh
git clone https://github.com/Kwapy/Study-timer-svelte

cd Study-timer-svelte

# install using NPM
npm install
```

2. Start the app

```
npm run dev
```
